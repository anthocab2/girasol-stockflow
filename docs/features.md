# Features

## MVP Features

The MVP focuses on the core inventory and sales process.

### Employee Features

- Log in using a punch number
- View employee name after login
- View available products
- Register a product sale
- Select product quantity
- Enter room number
- Select payment method
- Save sale with date and time
- View personal sales history

### Admin Features

- View all sales records
- View all employees
- View all products
- View current inventory
- Add new products
- Edit product stock
- Review inventory movements

## Payment Methods

The first version will support:

- Cash
- ATH
- Card
- Other

## Product Categories

The first version will support:

- Beverage
- Snack
- Alcohol
- Hygiene
- Other
